HTML INTRODUÇÃO

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções.

Então, o que é HTML?

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante. Aula 06

html Style

Em HTML, os estilos são usados para controlar a aparência e o layout dos elementos em uma página da web. Os estilos são definidos usando a linguagem CSS (Cascading Style Sheets), que permite especificar propriedades como cor, tamanho da fonte, margens, preenchimento e muito mais.

Existem diferentes maneiras de aplicar estilos em HTML:

  1. Estilos internos: Os estilos internos são definidos dentro do próprio documento HTML usando a tag `<style>`. Os estilos internos são especificados no cabeçalho do documento e afetam todos os elementos dentro desse documento. Eles são aplicados usando seletores de elementos, IDs ou classes.
  2. Estilos embutidos: Os estilos embutidos são aplicados diretamente a um elemento HTML usando o atributo `style`. Os estilos embutidos afetam apenas o elemento em que estão definidos e substituem quaisquer estilos definidos externamente.

    3. veja o exemplo :

  3. Folhas de estilo externas: As folhas de estilo externas são arquivos CSS separados que contêm todas as regras de estilo para um documento HTML. Esses arquivos CSS são vinculados ao documento HTML usando a tag `<link>` no cabeçalho do documento. As folhas de estilo externas permitem reutilizar estilos em vários documentos HTML e facilitam a manutenção e atualização dos estilos.
  4. Ao aplicar estilos, você pode usar seletores para direcionar elementos específicos. Os seletores podem ser baseados em elementos HTML, classes, IDs, atributos e muito mais. Você pode definir várias propriedades de estilo para cada seletor, como cor, tamanho da fonte, margens, preenchimento, bordas e outros.
  5. Em resumo, estilos em HTML são usados para controlar a aparência dos elementos em uma página da web, permitindo que você personalize cores, fontes, tamanhos e layout. O CSS é a linguagem usada para definir os estilos, e eles podem ser aplicados internamente, embutidos ou externamente por meio de folhas de estilo.

Aula 07

HTML Cores

Em HTML, as cores são usadas para definir a aparência visual dos elementos em uma página da web. As cores podem ser aplicadas a texto, fundos, bordas e outros elementos HTML. Existem várias formas de especificar cores em HTML, incluindo:

  1. Nomes de cores pré-definidos: HTML possui uma lista de nomes de cores pré-definidos que podem ser usados diretamente. Alguns exemplos comuns são "vermelho" (red), "azul" (blue), "verde" (green), entre outros. Esses nomes de cores são escritos em letras minúsculas.
  2. Valores hexadecimais: Os valores hexadecimais são uma forma de especificar cores usando uma combinação de números e letras. Eles começam com o símbolo "#" seguido por seis dígitos hexadecimais. Os dois primeiros dígitos representam a intensidade de vermelho (RR), os dois seguintes representam a intensidade de verde (GG) e os dois últimos representam a intensidade de azul (BB). Por exemplo, "#FF0000" representa vermelho, "#00FF00" representa verde e "#0000FF" representa azul.
    3. 3. Valores RGB: Os valores RGB são outra forma de especificar cores em HTML. RGB significa Red (vermelho), Green (verde) e Blue (azul). Os valores RGB são expressos como "rgb(R, G, B)", onde R, G e B são números inteiros entre 0 e 255 que representam a intensidade de cada componente de cor. Por exemplo, "rgb(255, 0, 0)" representa vermelho, "rgb(0, 255, 0)" representa verde e "rgb(0, 0, 255)" representa azul.

    Em resumo, as cores em HTML são usadas para definir a aparência visual dos elementos em uma página da web. Existem várias formas de especificar cores, incluindo nomes de cores pré-definidos, valores hexadecimais e valores RGB. A escolha do método depende das suas necessidades e preferências de design.

A tag <section> em HTML é um elemento de bloco que representa uma seção tematicamente relacionada de conteúdo em um documento. Ela é usada para dividir e organizar o conteúdo em partes distintas e significativas, permitindo uma melhor estruturação e compreensão do documento. Botões de submit

Para adicionar um botão de submit em um formulário, podemos utilizar o elemento input com type igual a submit. Esse tipo de botão envia os dados do formulário para o Web Server. Os textos desses botões são definidos com o atributo value.

Outra forma de adicionar um botão de submit em um documento HTML é utilizar o elemento button com type igual a submit. Diferentemente do elemento input, o elemento button permite a criação de botões com imagens além de texto.

Aula 16

Checkboxes e Radios

Para adicionar um checkbox em um formulário, devemos utilizar o elemento input com type igual a checkbox. Ao utilizar esse componente, é importante definir um valor para o atributo value. No envio do formulário, esse valor é transmitido ao Web Server se o checkbox correspondente estiver marcado.

Eventualmente é interessante agrupar um determinado conjunto de checkboxes. Por exemplo, considere um formulário que coleta as linguagens de programação que os usuários conhecem. Para cada linguagem, podemos definir um checkbox. Para agrupar esses checkboxes, basta definir o atri- buto name com o mesmo valor para eles.

Checkboxes Para adicionar um radio em um formulário, devemos utilizar o elemento input com type igual a radio. Ao utilizar esse componente, é importante definir um valor para o atributo value. No envio do formulário, esse valor é transmitido ao Web Server se o radio correspondente estiver marcado.

Eventualmente é interessante agrupar um determinado conjunto de radios. Por exemplo, considere um formulário que coleta o time preferido dos usuários. Para cada time, podemos definir um radio. Para agrupar esses radios, basta definir o atributo name com o mesmo valor para eles.

Radios

Por padrão, quando um formulário é exibido para os usuários, os checkboxes e os radios não estão marcados. Algumas vezes, desejamos que determinados checkboxes e radios estejam marcados quando os formulários são apresentados aos usuários. Para resolver esse problema, podemos utilizar o atributo checked do elemento input. Esse atributo não precisa de valor.

Para adicionar esse tipo de com- ponente, devemos utilizar o elemento select. Normalmente, o select é definido pelos navegadores como inline-level element.

As opções devem ser definidas no conteúdo do elemento select e elas são adicionadas com o ele- mento option. O conteúdo do elemento option é exibido para os usuários. Esse elemento possui um atributo chamado value. Quando o formulário for enviado, o valor do atributo value é transmitido ao Web Server se a opção correspondente foi selecionada pelo usuário. Se esse atributo não estiver definido, o conteúdo do elemento option é transmitido ao Web Server se a opção correspondente foi selecionada pelo usuário.

Por padrão, apenas um item de um drop-down list pode ser selecionado pelos usuários. Mas, utilizando o atributo multiple, um ou mais itens podem ser selecionados. Geralmente, mantendo a tecla CTRL pressionada, os usuários poderão escolher dois ou mais itens.

Nos drop-down lists com muitas opções, é interessante agrupar as opções em categorias. Esse agrupamento pode ser realizado com o elemento optgroup. Esse elemento possui um atributo cha- mado label. O valor desse atributo é exibido no drop-down list.

Iframe

O elemento < iframe >. é usado em HTML para incorporar uma página web ou documento dentro de outra página. Essa tag permite exibir o conteúdo de um documento externo dentro de uma janela delimitada na página atual. Isso é útil, por exemplo, quando você deseja incorporar um vídeo do YouTube, um mapa do Google Maps ou um widget de mídia social em seu site.

Aqui está um exemplo básico de uso do elemento ` Nesse exemplo, usamos o atributo `src` para especificar o URL do documento externo que queremos incorporar. No caso acima, é um vídeo do YouTube. O atributo `width` define a largura do iframe em pixels, e o atributo `height` define a altura em pixels.

Você também pode ajustar outras opções, como definir uma largura ou altura em porcentagem, definir uma borda, permitir ou desabilitar rolagem, entre outros.

Lembre-se de que nem todos os sites permitem a incorporação de seu conteúdo por meio do < iframe >. . Alguns sites possuem políticas de segurança que impedem a exibição de seu conteúdo em outros domínios. Portanto, é importante verificar a documentação ou as diretrizes do site para garantir que a incorporação

seja permitida. Além disso, é importante considerar a responsividade ao usar < iframe >. Você pode ajustar as dimensões do < iframe >. e usar técnicas de design responsivo para garantir que o conteúdo incorporado seja exibido corretamente em diferentes dispositivos e tamanhos de tela.

Atributo `sandbox`: O atributo `sandbox` permite restringir as permissões do conteúdo exibido no < iframe >. Isso é útil para garantir a segurança, impedindo que o conteúdo incorporado acesse recursos perigosos ou execute scripts maliciosos. Atributo `frameborder`: O atributo `frameborder` especifica se a borda ao redor do < iframe > deve ser exibida ou não. Defina o valor como `0` para não exibir a borda e `1` para exibir a borda.

Atributo `allowfullscreen`: O atributo `allowfullscreen` permite que o conteúdo do < iframe > seja exibido em modo de tela cheia quando ativado. Isso é comumente usado para vídeos incorporados.
Atributo `name`: O atributo `name` é usado para fornecer um nome ao < iframe >. Isso pode ser útil para referenciar o < iframe > em JavaScript ou em outros elementos da página.
Essas são algumas opções e atributos adicionais que você pode utilizar com o elemento < iframe > em HTML. Lembre-se de sempre verificar a documentação e as diretrizes dos serviços ou sites que você está incorporando para garantir o uso correto e adequado do `< iframe >.
