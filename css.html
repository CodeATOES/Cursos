Aula 1

Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem de folha de estilo utilizada para descrever a aparência e formatação de um documento escrito em HTML (Hypertext Markup Language). É comumente usado para controlar a aparência visual de páginas da web e interfaces de usuário.

O CSS funciona selecionando elementos HTML e aplicando estilos a eles. Ele permite que você modifique vários aspectos dos elementos, como tamanho, cor, fonte, espaçamento e posicionamento. Os estilos podem ser aplicados diretamente aos elementos HTML usando estilos embutidos (inline styles), ou podem ser definidos em um arquivo CSS separado e vinculado ao documento HTML.

Aqui está um exemplo de código CSS que define a cor e o tamanho da fonte de todos os parágrafos em um documento HTML:

p { color: blue; font-size: 16px; }
No exemplo acima, o seletor CSS "p" é usado para direcionar todos os elementos `

` no documento HTML. A propriedade `color` é definida como "blue" e a propriedade `font-size` é definida como "16px".

O CSS oferece uma ampla variedade de propriedades e valores que podem ser usados para estilizar elementos HTML. Algumas propriedades CSS comuns incluem:

- `color`: Define a cor do texto. - `font-size`: Define o tamanho da fonte. - `background-color`: Define a cor de fundo de um elemento. - `margin`: Define a margem ao redor de um elemento. - `padding`: Define o preenchimento dentro de um elemento. - `border`: Define as propriedades da borda de um elemento. - `width` e `height`: Definem as dimensões de um elemento. - `display`: Especifica como um elemento deve ser exibido.

Esses são apenas alguns exemplos, e existem muitas outras propriedades e valores CSS disponíveis para diferentes propósitos de estilização.

O CSS também suporta vários seletores que permitem direcionar elementos específicos ou grupos de elementos. Por exemplo, você pode usar seletores de classe, seletores de ID, seletores de atributo e pseudo-classes para aplicar estilos a elementos específicos com base em determinados critérios.

Em resumo, o CSS é uma ferramenta poderosa para controlar a apresentação visual de páginas da web e criar interfaces de usuário atraentes na web.

Aula 6

Mais sobre seletores

Os seletores CSS são usados para selecionar os elementos HTML nos quais você deseja aplicar estilos. Existem vários tipos de seletores CSS que você pode usar para segmentar elementos de maneiras diferentes. Aqui estão alguns dos seletores mais comuns:

Seletor de elemento: Seleciona elementos HTML com base em seus nomes de tag. Por exemplo, o seletor `p` seleciona todos os elementos <p> no documento.



Seletor de classe: Seleciona elementos HTML com base em suas classes atribuídas. O seletor de classe é denotado pelo prefixo `.` seguido pelo nome da classe. Por exemplo, o seletor `.destaque` seleciona todos os elementos que possuem a classe "destaque".


Seletor de ID: Seleciona um único elemento HTML com base em seu atributo "id". O seletor de ID é denotado pelo prefixo `#` seguido pelo nome do ID. Por exemplo, o seletor `#meu-elemento` seleciona o elemento que possui o ID "meu-elemento".


Seletor de descendência: Seleciona elementos HTML com base em sua hierarquia no documento. O seletor de descendência é denotado por espaços entre os seletores.

Por exemplo, o seletor `div p` seleciona todos os elementos <p> que estão dentro de um elemento <div>.


Seletor de filho direto: Seleciona elementos HTML que são filhos diretos de outro elemento. O seletor de filho direto é denotado pelo caractere `>` entre dois seletores. Por exemplo, o seletor `div > p` seleciona todos os elementos <p> que são filhos diretos de um elemento <div>.


Seletor de atributo: Seleciona elementos HTML com base em seus atributos. Você pode especificar o nome do atributo e, opcionalmente, o valor do atributo. Por exemplo, o seletor `[type="text"]` seleciona todos os elementos que possuem o atributo `type` com o valor "text".

Selectores

7. Seletor de pseudo-classe: Seleciona elementos HTML com base em um estado ou interação específica. As pseudo-classes são denotadas pelo uso de dois pontos `:` antes do nome da pseudo-classe. Por exemplo, o seletor `a:hover` seleciona os links quando o cursor do mouse está sobre eles.

8. Seletor de pseudo-elemento: Seleciona partes específicas de um elemento HTML. Os pseudo-elementos também são denotados pelo uso de dois pontos `:` antes do nome do pseudo-elemento. Por exemplo, o seletor `::first-line` seleciona a primeira linha de um bloco de texto.


Seletor de adjacente: Seleciona o elemento imediatamente adjacente a outro elemento. O seletor de adjacente é denotado pelo caractere `+` entre os dois seletores. Por exemplo, o seletor `h2 + p` seleciona o elemento <p> que é imediatamente após um elemento <h2>.


Seletor de irmão geral: Seleciona todos os elementos irmãos que seguem um determinado elemento. O seletor de irmão geral é denotado pelo uso de um til `~` entre os dois seletores. Por exemplo, o seletor `p ~ span` seleciona todos os elementos <span> que seguem um elemento <p>.
Seletor de negação: Seleciona elementos que não correspondem a um determinado seletor. O seletor de negação é denotado pelo uso de `:not()` seguido pelo seletor que você deseja excluir. Por exemplo, o seletor `p:not(.destaque)` seleciona todos os elementos `

` que não têm a classe "destaque".

Seletor de atributo avançado: Permite selecionar elementos com base em padrões de atributos mais complexos. Alguns exemplos incluem: - `[atributo^=valor]`: Seleciona elementos com um atributo que começa com um determinado valor. - `[atributo$=valor]`: Seleciona elementos com um atributo que termina com um determinado valor. - `[atributo*=valor]`: Seleciona elementos com um atributo que contém um determinado valor em qualquer lugar.

Seletor de grupo: Permite agrupar vários seletores em uma única regra. Os seletores são separados por vírgulas. Por exemplo, o seletor `h1, h2, h3` seleciona todos os elementos <h1>, <h2> e <h3>.
Seletor de primeiro filho: Seleciona o primeiro elemento filho de um elemento pai. O seletor de primeiro filho é denotado por `:first-child`. Por exemplo, o seletor `li:first-child` seleciona o primeiro elemento <li> em uma lista.


Seletor de último filho: Seleciona o último elemento filho de um elemento pai. O seletor de último filho é denotado por `:last-child`. Por exemplo, o seletor `li:last-child` seleciona o último elemento <li> em uma lista.



Seletor de primeiro elemento de tipo: Seleciona o primeiro elemento de um determinado tipo dentro de seu elemento pai. O seletor de primeiro elemento de tipo é denotado por `:first-of-type`. Por exemplo, o seletor `p:first-of-type` seleciona o primeiro elemento <p> dentro de seu elemento pai.


Seletor de último elemento de tipo: Seleciona o último elemento de um determinado tipo dentro de seu elemento pai. O seletor de último elemento de tipo é denotado por `:last-of-type`. Por exemplo, o seletor `p:last-of-type` seleciona o último elemento <p> dentro de seu elemento pai.


Seletor de elemento vazio: Seleciona elementos que não têm nenhum filho. O seletor de elemento vazio é denotado por `:empty`. Por exemplo, o seletor `div:empty` seleciona todos os elementos <div> que não têm conteúdo.

Seletor de atributo com prefixo: Seleciona elementos com um atributo que começa com um determinado valor. O seletor de atributo com prefixo é denotado por `^=`. Por exemplo, o seletor `[atributo^=valor]` seleciona elementos em que o valor do atributo começa com "valor".

Seletor de atributo com correspondência exata: Seleciona elementos com um atributo que corresponde exatamente a um determinado valor. O seletor de atributo com correspondência exata é denotado por `=`. Por exemplo, o seletor `[atributo=valor]` seleciona elementos em que o valor do atributo é exatamente "valor".

Seletor de link ativo: Seleciona o link ativo, que é o link em que o usuário está interagindo no momento. O seletor de link ativo é denotado por `:active`. Por exemplo, o seletor `a:active` seleciona o link que está sendo clicado.


Seletor de link hover: Seleciona links quando o cursor do mouse está sobre eles. O seletor de link hover é denotado por `:hover`. Por exemplo, o seletor `a:hover` seleciona os links quando o cursor do mouse está sobre eles.


Seletor de link focus: Seleciona links que estão em foco, ou seja, quando são selecionados por meio de teclado ou outro método de navegação. O seletor de link focus é denotado por `:focus`. Por exemplo, o seletor `a:focus` seleciona os links que estão em foco.

    Opacidade

    Opacidade refere-se à propriedade de um objeto ou elemento gráfico de ser transparente ou opaco. Em CSS, a opacidade é controlada pela propriedade `opacity`, que aceita valores de 0 a 1, sendo 0 totalmente transparente e 1 totalmente opaco. Portanto, definir a opacidade de um elemento para 0.5 significa que ele estará 50% transparente.

    A notação RGBA (Red, Green, Blue, Alpha) é uma extensão da notação RGB para representar cores com transparência. Enquanto a notação RGB usa três valores (de 0 a 255) para especificar a intensidade de vermelho, verde e azul de uma cor, a notação RGBA adiciona um quarto valor (de 0 a 1) para representar a opacidade.

    Na notação RGBA, os três primeiros valores representam a intensidade de vermelho, verde e azul, respectivamente, variando de 0 a 255 ou de 0% a 100%. O quarto valor, chamado de canal alfa, representa a opacidade e varia de 0 a 1, onde 0 é totalmente transparente e 1 é totalmente opaco.

    Portanto, ao definir uma cor usando a notação RGBA, você pode controlar a opacidade adicionando um valor de 0 a 1 após os valores de vermelho, verde e azul.

    Espero que isso esclareça a definição de opacidade e a notação RGBA. Se você tiver mais dúvidas, fique à vontade para perguntar.

    Você pode ajustar o valor da propriedade `opacity` para controlar a opacidade desejada, variando de 0 (totalmente transparente) a 1 (totalmente opaco). Valores intermediários, como 0.5, fornecerão um efeito de transparência proporcional.

    Lembre-se de que a propriedade `opacity` afeta não apenas o conteúdo do elemento, mas também todos os elementos filhos. Ou seja, se você definir a opacidade de um elemento pai, ela será aplicada a todos os elementos filhos dentro dele.

    hsl, HSLA



    HSL (Matiz, Saturação, Luminosidade): - Matiz (Hue): Representa a cor em um círculo cromático. O valor varia de 0 a 360, onde 0 e 360 correspondem ao vermelho, 120 ao verde e 240 ao azul. Os outros valores entre 0 e 360 representam cores intermediárias. - Saturação (Saturation): Controla a intensidade da cor. O valor varia de 0% a 100%, onde 0% significa sem saturação (tons de cinza) e 100% corresponde à saturação máxima. - Luminosidade (Lightness): Determina o brilho da cor. O valor varia de 0% a 100%, onde 0% é preto e 100% é branco.

    HSLA (Matiz, Saturação, Luminosidade, Alfa): - Alfa (Alpha): É um valor opcional que controla a transparência da cor. O valor varia de 0 a 1, onde 0 significa totalmente transparente e 1 corresponde à opacidade total. Com o valor de alfa, você pode controlar a opacidade da cor e permitir que o conteúdo subjacente seja visível através dela.

    Ao usar a notação HSL ou HSLA em CSS, você pode especificar cores de forma mais intuitiva e ter controle sobre a matiz, saturação, luminosidade e transparência de uma cor.

    Aqui estão alguns exemplos de notação HSL e HSLA: - `hsl(0, 100%, 50%)`: Vermelho puro, com saturação máxima e luminosidade de 50%. - `hsl(120, 50%, 75%)`: Verde com saturação de 50% e luminosidade de 75%. - `hsla(240, 100%, 50%, 0.5)`: Azul com saturação máxima e luminosidade de 50%, com transparência de 50%.

    Essas notações fornecem uma maneira flexível e expressiva de especificar cores em CSS, permitindo que você ajuste a matiz, saturação, luminosidade e transparência para alcançar os efeitos visuais desejados.

    Espero que isso esclareça a definição das notações HSL e HSLA! Se você tiver mais dúvidas, estou aqui para ajudar.
